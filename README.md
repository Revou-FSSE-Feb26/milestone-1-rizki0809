## CSS Implementation

- Responsif Page : Ponsel (max-width: 400 px), Tablet (max-width: 768px), Desktop (max-width>768)
- Navbar : Flexbox, Hamburger button untuk tampilan page yang responsif. Hamburger button muncul pada tampilan ponsel dan tablet.
- Home : CTA yang mengarahkan visitor ke section about me.
- About me : Responsive page menggunakan flexbox direction.
- Project : Menggunakan grid untuk menampilkan projek, dengan tampilan projek 1 lebih besar dari projek lainnya. untuk tampilan desktop menggunakan grid dengan 3 kolom, tampilan tablet menggunakan grid dengan 2 kolom, dan tampilan ponsel menggunakan grid dengan 1 kolom.
- Contact : styling pada form dan button.
- Footer : styling text.
